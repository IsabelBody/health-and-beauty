## Pinterest fabric research

Search Pinterest for each fabric on a **straight / low-rise mini** (or drop-waist dress). Only save pins that pass the checklist.

| Fabric | Search ideas | Save if… |
|--------|--------------|----------|
| **Structured denim** | `low rise denim mini skirt straight`, `black rigid denim mini`, `dark wash structured mini skirt outfit` | Straight column, contrast stitch, dark wash; not skater / limp stretch |
| **Matte leather** | `black leather mini skirt outfit`, `leather mini skirt structured tank` | Matte black, pencil/straight; tonal head-to-toe |
| **Wool gabardine / suiting** | `gabardine mini skirt`, `tailored black mini skirt wool`, `charcoal pencil mini skirt` | Clean pressed line; not bodycon |
| **Cotton twill / drill** | `black twill mini skirt`, `chino mini skirt outfit` | Crisp matte; low-rise or clean waistband |
| **Ponte** | `ponte knit pencil mini skirt`, `structured knit mini skirt black` | Thick knit holding shape — not jersey tube |
| **Drop-waist crepe / sateen** | `drop waist mini dress black`, `sateen mini dress square neck` | One-piece midriff; structured fabric |

**Tops (structured tank / corset / layer):**

| Top type | Search ideas | Save if… |
|----------|--------------|----------|
| **Structured tank** | `square neck tank black outfit`, `ponte tank mini skirt`, `halter top low rise mini` | Thick straps; ponte/rib/cotton holds line; square or V neck |
| **Corset + mini** | `underbust corset mini skirt`, `structured corset black outfit` | Boning, panels; black/oxblood; not lace cottagecore |
| **Drop-waist dress** | `drop waist mini dress square neck black` | One-piece; crepe/sateen; bare waist built in |
| **Jacket + mini** | `blazer mini skirt outfit black`, `structured jacket mini skirt tank` | Hip-length blazer; ensemble read |
| **Turtleneck + mini** | `black turtleneck mini skirt boots` | Fitted sleek knit; not slouch |

**Reject reference:** also search `jersey bodycon mini skirt`, `skater mini skirt`, `oversized sweater mini skirt` — note what **not** to save (cling, flare, slouch volume).

## Pinterest save checklist

Quick gate: if a pin fails **any** “Don’t save” rule, skip it. If it passes avoids and hits at least one “Save” signal in the relevant category, it’s worth pinning.

### Miniskirts & shorts

| Save | Don’t save |
|------|------------|
| Straight, pencil, slim column mini or short | Tiered, ruffled, skater / circle, tulle |
| Flat hip → upper thigh; stitched-down pleats only | Gathered waist, volume at hip, wide A-line |
| **Tier 1:** structured denim, matte leather/faux leather | Stretch bodycon jersey; floaty unstructured fabric |
| **Tier 2:** wool gabardine, suiting, cotton twill/drill | Skater/circle cut (any fabric) |
| **Tier 3:** ponte straight mini; evening crepe/sateen drop-waist | High-low hem; ultra flimsy poly; super-stretch jegging denim |
| Low-rise or drop-waist line | Y2K extreme hip-slung (unless whole look is still sharp / ensemble) |

### Tops (tank / corset / layering)

| Save | Don’t save |
|------|------------|
| Square, V, or scoop neck; halter; thick straps | Boat neck (midriff formulas); spaghetti-only flimsy cami |
| Structured rib, ponte, crisp cotton, boning / panels | Peasant, cowl, oversized slouch; cottagecore lace corsets |
| Underbust corset over fitted tank; drop-waist dress (one-piece) | Lace romantic corset; pastel ruffles / ribbon bows |
| Corset meets skirt with tiny gap, or true drop-waist dress | Micro crop + micro mini together (daily board) |

### Silhouette & Kibbe (DC)

| Save | Don’t save |
|------|------------|
| Triangular read — shoulder / neck emphasis, tailored chic | Soft, flouncy, festival, thrown-on |
| Whole outfit reads as **ensemble** (planned head-to-toe) | Random separates with no color story |
| Tonal top → skirt → shoes when midriff is out | Loud unrelated color block **at natural waist** + bare stomach |
| Jacket over mini (hip-length structured); long pendant / column scarf OK | Ultra-cropped bolero; waist-cinchers; “softened” matronly set |
| Moderate yang detail: contrast stitch, sharp lapels, geometric bag | All-sharp trendy with zero Classic buffer; gamine stacked crops |

### Color — deep winter ([[../wiki/me/face-blueprint]])

| Save | Don’t save |
|------|------------|
| Black, charcoal, navy, cool red, cool berry / raspberry, oxblood, crisp white | Orange, peach, coral, brick, warm terracotta, muddy warm tones |
| High contrast via **neckline, shoes, bag, lip** — repeated accents | One random accent that chops the outfit at the waist |
| Black-on-black mini uniform; charcoal + black + oxblood shoe | Pastel cottagecore palette |
| White + black **if** shoes or jacket ties the bottom half | Unrelated third color block at waist when stomach is out |

### Full-look pins (outfit photos)

**Save when:** mini formula + structured top + sleek shoe + intentional styling; skin-forward but polished.

**Don’t save when:** inspo is cute but breaks two or more rows above (e.g. skater mini + pastel lace corset + warm coral top).
