# Daily makeup routine

Target: thin layers; placement per [[../../wiki/me/face-blueprint]]. Products: [[../../owned-products-my-makeup-collection]].

## Skin prep (before makeup)

| Step | Product | Notes |
|------|---------|--------|
| Cleanse — T-zone | CeraVe Foaming Cleanser | Forehead (and T-zone if using split cleanse) |
| Cleanse — rest | CeraVe Hydrating Cleanser | Cheeks, jaw, etc. |
| Moisturise | CeraVe PM lotion **or** less on T-zone if oily | Less on forehead, nose, temples; more on dry cheeks. Wait until almost dry |
| Primer | Laura Mercier Pure Canvas | Thin layer; wait. T-zone: minimal product if oily by midday |

**T-zone oil:** Still oily by midday/end of day with split cleanse — see [[../../journal/2026-05-26-tzone-oil-matte-lips]]. Adjust primer/moisturiser/powder on forehead before buying new products.
